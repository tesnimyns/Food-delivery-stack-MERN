# FOOD DELIVERY - Application de Livraison

## Description
Ce projet est une **application de livraison** développée avec la stack **MERN (MongoDB, Express, React, Node.js)**.  
Il permet de gérer les commandes, les utilisateurs, les livraisons, et inclut un **chatbot intelligent** pour assister les clients.  

Le projet est structuré en **4 dossiers principaux** :  
1. **frontend/** : Interface utilisateur pour les clients (React).  
2. **backend/** : API REST pour la gestion des commandes, utilisateurs et livraisons (Node.js + Express + MongoDB).  
3. **admin/** : Tableau de bord administrateur pour gérer les commandes et utilisateurs (React/Admin).  
4. **chatbot/** : Assistant conversationnel développé avec **Dialogflow**, intégré à l’application.  

---

## Fonctionnalités

- Gestion des utilisateurs et authentification (JWT).  
- Suivi et gestion des commandes en temps réel.  
- Tableau de bord administrateur complet.  
- Chatbot intelligent pour répondre aux questions des clients.  
- Notifications en temps réel via Socket.io (optionnel).  

---

## Architecture du projet
```bash
project-root/
│
├── frontend/       # Interface utilisateur
├── backend/        # API et logique serveur
├── admin/          # Tableau de bord administrateur
└── chatbot/        # Intégration Dialogflow

```
